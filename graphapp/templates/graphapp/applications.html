AirliftCalc

Introduction

AirliftCalc is a flight booking and luggage management system that integrates web development with optimization techniques. It simplifies complex tasks using the Simplex Method, ensuring efficient cost calculations based on user's baggage weight.

How it Works

  • User Input – The system takes inputs like passenger weight, luggage weight, and flight details.
  • Optimization Process – It applies the Simplex Method to determine whether the luggage exceeds weight limits and adjusts costs accordingly.
  • Cost Calculation – If the total weight crosses a predefined threshold, additional charges are applied dynamically.
  • Booking Confirmation – After optimization, the system finalizes the booking with the appropriate fare.
